# Code of Conduct

## Our standards

We expect contributors to:

- act professionally and respectfully
- assume good intent while discussing technical disagreements
- focus feedback on code, behavior, and documented requirements
- avoid harassment, discrimination, personal attacks, and disruptive conduct

## Scope

This applies to project spaces, issues, pull requests, discussions, and other community interactions tied to this repository.

## Enforcement

Project maintainers may remove comments, reject contributions, or restrict participation for behavior that violates this code of conduct.

If you experience or witness unacceptable behavior, contact `@hnhoangdz` on GitHub with relevant context and links.
