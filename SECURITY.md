# Security Policy

## Supported versions

Security fixes are applied to the latest `1.x` release line.

## Reporting a vulnerability

- Do not open a public GitHub issue for suspected vulnerabilities.
- Use GitHub Security Advisories for private disclosure when available.
- If private reporting is not available, contact the maintainer ([@hnhoangdz](https://github.com/hnhoangdz)) directly before publishing details.

Please include:

- affected version
- impact summary
- reproduction steps
- any proof-of-concept or logs needed to validate the issue
